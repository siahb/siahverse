      📝 How to Use Siah To Do

      ×

      🚀 Getting Started

      1. Login: Click "🔐 Enter Password" and enter your admin password
      2. Add Tasks: Type in the main input field and press Enter
      3. Set Properties: Add tags, priority, due dates, and repeating schedules before pressing Enter

      ⌨️ Keyboard Shortcuts

      Search

      • Ctrl + F - Open search
      • / - Quick search
      • Esc - Close search

      Adding Tasks

      • Enter - Add task (from any input)
      • Tab - Navigate between fields

      🔍 Special Search Terms

      Use these special searches to quickly filter your tasks:

      • untagged or no-tags - Find tasks without tags
      • high or p:high - Show high priority tasks !!!
      • medium or p:medium - Show medium priority tasks !!
      • low or p:low - Show low priority tasks !
      • no-priority or p:none - Show tasks without priority

      📋 Task Management

      Task Actions

      • ✅ - Mark as done (or advance repeating tasks)
      • ✏️ - Edit task inline
      • ❌ - Delete task
      • ↩️ - Undo (restore deleted tasks)

      Bulk Actions

      • Select multiple tasks with checkboxes
      • Use "🗑️ Delete Selected" for bulk deletion
      • Select All checkbox in header

      📅 Scheduling & Repeating

      Due Dates

      • Set due dates with the date picker
      • Click "Today" button for quick date setting
      • Overdue tasks show Overdue pills

      Repeating Tasks

      • Daily: Repeats every X days
      • Weekly: Choose specific days of the week
      • When you complete a repeating task, it automatically advances to the next occurrence

      🏷️ Tags & Priority

      Tags

      • Add multiple tags separated by commas: work, urgent, meeting
      • Search by tag name to filter tasks
      • Use untagged search to find tasks that need tags

      Priority Levels

      • !!! High - Critical/urgent tasks (red)
      • !! Medium - Important tasks (orange)
      • ! Low - Nice to have tasks (green)

      📊 Sorting & Organization

      • Custom (drag): Drag and drop to reorder (click "↕️ Reorder" first)
      • Priority: High to low priority
      • Due date: Ascending or descending
      • A-Z: Alphabetical by task name
      • Newest: Most recently created first

      💡 Pro Tips

      • Use the untagged search to find and organize tasks without tags
      • Priority search terms help you focus: try high to see urgent tasks
      • The progress bar shows completion percentage and overdue task count
      • Done tasks are collapsible - click the "Marked as Done ✅" header
      • Drag reordering only works when sorted by "Custom (drag)"
